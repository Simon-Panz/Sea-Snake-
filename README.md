Hotkeys für das Spiel:
Esc: Im Startbildschirm schließen, vom Spiel in den Startbildschirm und vom Game Over-Screen ins Spiel
Space: Im Startbildschirm oder Game Over-Screen Spiel starten
Up: Nach oben bewegen
Left: Nach links bewegen
Down: Nach unten bewegen
Right: Nach rechts bewegen
E: Im Startbildschirm Schwierigkeit auf easy stellen (auch standartmäßig drin)
R: Im Startbildschirm Schwierigkeit auf hard stellen
1: Im Startbildschirm Spielerzahl auf 1 Stellen
2: Im Startbildschirm Spielerzahl auf 2 stellen (mit WASD und Pfeiltasten seperat steuerbar)
3. Im Startbildschirm Spielerzahl auf 2 stellen, zweite Schlange bewegt sich automatisch
